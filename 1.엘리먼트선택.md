# 엘리먼트 선택하기

## 태그 선택자로 엘리먼트 선택하기
```html
<div>
    <h1>제목1</h1>
    <p>단락1</p>
    <p>단락2</p>
    <p>단락3</p>
    <p>단락4</p>
    <p>단락5</p>
</div>
```

```javascript
<script>
    // 태그명이 h1인 모든 엘리먼트를 선택해서 삭제한다.
    $("h1").remove();
    // <h1>제목1</h1> 이 삭제된다.

    // 태그명이 p인 모든 엘리먼트를 선택해서 삭제한다.
    $("p").remove();
    // <p>단락1</p>, <p>단락2</p>, <p>단락3</p>, <p>단락4</p>, <p>단락5</p> 이 삭제된다
</script>
```

## 클래스 선택자로 엘리먼트 선택하기
```html
<div>
    <h1>제목1</h1>

    <div>
        <h3 class="intro text-danger">소제목1 입니다.</h3> 
        <p >단락1 입니다.</p>
    </div>

    <div>
        <p class="intro">단락2 입니다.</p>
        <p class="intro">단락3 입니다.</p>
        <p>단락4 입니다.</p>
    </div>
</div>
```

```javascript
<script>
    // class="intro" 클래스 속성값에 intro를 포함하고 있는 모든 엘리먼트를 선택해서 삭제한다.
    $(".intro").remove();
    // 위의 코드는 <h3>소제목1</h3>, <p>단락2</p>, <p>단락3</p> 태그를 삭제한다. 

    // 태그명이 p인것 중에서class="intro" 클래스 속성값에 intro를 포함하고 있는 모든 엘리먼트를 선택해서 삭제한다.
    $("p.intro").remove();
    // 위의 코드는 <p>단락2</p>, <p>단락3</p> 태그를 삭제한다. 
</script>
```

## 아이디 선택자로 엘리먼트 선택하기
```html
<form>
    이름 : <input type="text" name="name" id="user-name" />
    메일 : <input type="text" name="email" id="user-email" /> 
</form>
```

```javascript
<script>
    // .val(값) 함수는 선택된 입력필드 엘리먼트의 값을 지정된 값으로 변경한다.
    // .val() 함수는 선택된 입력필드 엘리먼트의 입력값을 조회해서 반환한다.

    // id="user-name"인 엘리먼트를 선택하고, 입력필드의 값을 "홍길동"으로 변경한다.
    $("#user-name").val("홍길동");

    // id="user-email"인 엘리먼트를 선택하고, 입력값을 조회해서 반환한다.
    let 이메일주소 = $("#user-email").val();
    
</script>
```

## 자손 선택자로 엘리먼트 선택하기
```html
<body>
    <div>
        <h3>참가자</h3>
        <ul>
            <li>김유신</li> <!-- 삭제됨-->
            <li>강감찬</li> <!-- 삭제됨-->
            <li>이순신</li> <!-- 삭제됨-->
        </ul>
    </div>

    <h3>관계기관</h3>
    <ul>
        <li>도로공사</li> <!--삭제되지 않음 -->
        <li>한국은행</li> <!--삭제되지 않음 -->
        <li>도시철도</li> <!--삭제되지 않음 -->
    </ul>
</body>
```

```javascript
<script>
    // div 엘리먼트의 자손 엘리먼트 중에서 태그명이 li인 모든 엘리먼트를 선택해서 삭제한다.
    $("div li").remove();
    // <li>도로공사</li>, <li>한국은행</li>, <li>도시철도</li>는 div 엘리먼트의 자손이 아니기 때문에 삭제되지 않는다.
</script>
```

## 자식 선택자로 엘리먼트 선택하기
```html
<body>
    <div>
        <h3>참가자</h3>
        <a href="">링크1</a> <!-- 삭제됨 -->

        <ul>
            <li>
                <a href="">링크2</a> <!--삭제되지 않음 -->
            </li>
            <li>
                <a href="">링크3</a> <!--삭제되지 않음 -->
            </li>
            <li>
                <a href="">링크4</a> <!--삭제되지 않음 -->
            </li>
        </ul>
    </div>
</body>
```

```javascript
<script>
    // div 엘리먼트의 자식 엘리먼트 중에서 태그명이 a인 모든 엘리먼트를 선택해서 삭제한다.
    $("div > a").remove();
    // <a>링크1</a> 만 삭제된다.
    // <a>링크2</a>, <a>링크3</a>, <a>링크4</a> 태그는 div의 자식이 아닌 자손이다.

    $("div a").remove();
    // <a>링크1</a>, <a>링크2</a>, <a>링크3</a>, <a>링크4</a>가 삭제된다.
</script>
```

## 순서관련 필터 선택자로 엘리먼트 선택하기
```html
<div>
    <p>단락1</p> <!-- 0번째 -->
    <p>단락2</p> <!-- 1번째 -->
    <p>단락3</p> <!-- 2번째 -->
    <p>단락4</p> <!-- 3번째 -->
    <p>단락5</p> <!-- 4번째 -->
    <p>단락6</p> <!-- 5번째 -->
    <p>단락7</p> <!-- 6번째 -->
</div>
```

```javascript
<script>
    // p엘리먼트 중에서 첫번째 엘리먼트를 선택한다.
    $("p:first").remove();      // <p>단락1</p>이 선택되어 삭제된다.
    // p엘리먼트 중에서 마지막번째 엘리먼트를 선택한다.
    $("p:last").remove();       // <p>단락7</p>이 선택되어 삭제된다.

    // p엘리먼트 중에서 0번째 엘리먼트를 선택한다.
    $("p:eq(0)").remove();      // <p>단락1</p>이 선택되어 삭제된다.
    // p엘리먼트 중에서 1번째 엘리먼트를 선택한다.
    $("p:eq(1)").remove();      // <p>단락2</p>이 선택되어 삭제된다.
    // p엘리먼트 중에서 2번째 엘리먼트를 선택한다.
    $("p:eq(2)").remove();      // <p>단락3</p>이 선택되어 삭제된다.
    // p엘리먼트 중에서 3번째 엘리먼트를 선택한다.
    $("p:eq(3)").remove();      // <p>단락4</p>이 선택되어 삭제된다.
    // p엘리먼트 중에서 6번째 엘리먼트를 선택한다.
    $("p:eq(6)").remove();      // <p>단락7</p>이 선택되어 삭제된다.

    // p엘리먼트 중에서 3번째보다 이전 엘리먼트를 선택한다.
    $("p:lt(3)").remove();      // <p>단락1, <p>단락2</p>, <p>단락3</p>이 삭제된다.
    // p엘리먼트 중에서 3번째보다 이후 엘리먼트를 선택한다.
    $("p:gt(3)").remove();      // <p>단락5</p>, <p>단락6</p>, <p>단락7</p>이 삭제된다.    
</script>
```

## 자식 필터 선택자
```html
<body>
    <div>
        <p>단락1</p>
        <p>단락2</p>
        <p>단락3</p>
        <p>단락4</p>
    </div>
    
    <div>
        <p>단락5</p>
        <p>단락6</p>
        <p>단락7</p>
    </div>
    
    <div>
        <p>단락8</p>
        <p>단락9</p>
        <p>단락10</p>
        <p>단락11</p>
    </div>
</body>
```

```javascript
<script>
    // 순서관련 필터 선택자와 자식 필터  선택자는 구분된다.

    // 순서관련 필터 선택자 //////////////////////////////////////////////////////////////////////////////
    // p중에서 첫번째 엘리먼트가 선택된다.
    // 무조건 하나만 선택된다.
    $("p:first").text("선택됨");    // <p>단락1</p>이 선택되고 내용이 "선택됨"으로 변경된다.

    // p중에서 마지막번째 엘리먼트가 선택된다.
    // 무조건 하나만 선택된다.
    $("p:last").text("선택됨");     // <p>단락11</p>이 선택되고 내용이 "선택됨"으로 변경된다.

    // 자식 필터 선택자 //////////////////////////////////////////////////////////////////////////////
    // p중에서 부모 기준으로 첫번째 자식인 p를 전부 선택한다.
    // 여러 개 선택될 수 있다.
    $("p:first-child").text("선택됨");  //  <p>단락1</p>,  <p>단락5</p>,  <p>단락8</p>이  선택되고 내용이 "선택됨"으로 변경된다.

    // p중에서 부모 기준으로 마지막번째 자식인 p를 전부 선택한다.
    // 여러 개 선택될 수 있다.
    $("p:last-child").text("선택됨");  //  <p>단락4</p>,  <p>단락7</p>,  <p>단락11</p>이  선택되고 내용이 "선택됨"으로 변경된다.

    // p중에서 부모 기준으로 두번째 자식인 p를 전부 선택한다.
    $("p:nth-child(2)").text("선택됨")  // <p>단락2</p>, <p>단락6</p>, <p>단락9</p>이 선택되고 내용이 "선택됨"으로 변경된다.
</script>
```
